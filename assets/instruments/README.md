# Bundled Instrument Samples

These folders contain compact, curated multisample maps for the built-in virtual keyboard.
The app uses these through the native C++/Tracktion sampler path; JavaScript only sends
the sample-region manifest to the engine.

## Piano

- Instrument: Splendid Grand Piano
- Source: https://github.com/sfzinstruments/SplendidGrandPiano
- License: Public Domain
- Bundled subset: selected forte FLAC notes from `Samples/`

## Bass

- Instrument: Growlybass by Karoryfer Samples
- Source: https://github.com/sfzinstruments/karoryfer.growlybass
- License: CC0-1.0
- Bundled subset: selected sustained direct-recorded WAV notes from `sustain/`

## Guitar

- Instrument: Emilyguitar by Karoryfer Samples
- Source: https://github.com/sfzinstruments/karoryfer.emilyguitar
- License: CC0-1.0
- Bundled subset: selected direct-recorded WAV notes from `notes/`
